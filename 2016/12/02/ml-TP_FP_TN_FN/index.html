Machine Learning小知识之TP-FP-TN-FN

发表于 2016-12-02 | 分类于 Machine Learning |

对于一个二分类问题，就结果而言，会出现下面四种情况：

  • 实际为正样本：Postive(P)
  • 实际为负样本：Negative(N)
  • 事实为正样本，也被判定为正样本：True Positive(TP)，正确划分为正类
  • 事实为正样本，却被判定为负样本：False Negative(FN)，错误划分为负类
  • 事实为负样本，也被判定为负样本：True Negative(TN)，正确划分为负类
  • 事实为负样本，却被判定为正样本：False Positive(FP)，错误划分为正类

根据这四种情况，定义如下的几种评价指标：

  1. 正确率(accuracy)

    accuracy = (TP + TN) / (P+N)，表示所有样本中都被正确分类的样本所占的比率，正确率越高，分类器越好；

  2. 错误率(error rate)

    error rate = (FP + FN) / (P + N)，与正确率相对应，accuracy = 1 - erros rate；

  3. 灵敏度(sensitive,TPR)

    sensitive = TP / P，表示所有正样本中被正确划分的比例，衡量分类器对正样本的识别能力，也被称为真正类率(True Positive Rate, TPR);

  4. 负正类率(False Positive Rate, FPR)

    FPR = FP / (FP + TN), 计算的是分类器错认为正样本的负实例占所有负实例的比例。

  5. 特效度(specificity,TNR)

    specitive = TN / N = 1 - FPR，表示所有负样本被正确划分的比例，衡量分类器对负样本的识别能力，也被称为真负类率(True Negative Rate，TNR)；

  6. miss rate(False Negative Rate, FNR)

    miss rate = FN / (FN + TP)，表示分类器错认为负样本的正实例占所有正实例的比例。

  7. 精度(precision)

    precision = TP / (TP + FP)，表示被分为正样本中实际为正样本的比例；

  8. 召回率(recall)

    recall = TP / (TP + FN) = TP / P = sensitive，表示有多少个正样本被正确分类；
